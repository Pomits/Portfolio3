HTML(Hypertext Markup Language), es un estándar utilizado para la creación y el diseño de páginas web. Es un lenguaje de marcado que se usa para estructurar el contenido de una página web mediante la utilización de etiquetas y atributos. Es como si fuera el esqueleto del proyecto

CSS, Hojas de Estilo en Cascada (Cascading Style Sheets), es un lenguaje utilizado para definir el estilo y la presentación de documentos HTML. Su objetivo principal es separar la estructura del contenido (definida por HTML) de su presentación visual. Es como la cara del proyecto

JavaScript (JS) es un lenguaje de programación de alto nivel que se utiliza para agregar interactividad y dinamismo a las páginas web. Es como el musculo del proyecto
